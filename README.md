### Prerequisite

- Install [Java 15+](https://www.java.com/en/)
- Install [Gradle](https://gradle.org/)

### Install

- To build and run the editor, you must download the source code and run with an IDE

  ```bash
  git clone https://github.com/radiumgame/radium.git
  ```
